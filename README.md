# sns_app

## 概要
某snsアプリを再現したWebアプリケーションです。 
DBに情報を追加、編集、削除までできるようにしました。

---

## 使用技術
- HTML
- CSS
- JavaScript
- React
- node
- MySQL

---

## アプリ内容・機能
- ログイン・ログアウト機能
- いいね機能
- 新規投稿機能
- 投稿削除機能
- コメント機能

---

## 工夫した点
- UIをコンポーネント化することでデザインの統一とコード量を減らすことを意識しました
